# Análisis de Congestión de Tráfico en Gran Bretaña 

Este proyecto utiliza técnicas de Big Data y Machine Learning para analizar y predecir la congestión vehicular en carreteras de Gran Bretaña.

## 1. Obtención del Dataset
Debido al tamaño de los archivos, los datos no están incluidos en el repositorio. Sigue estos pasos para obtenerlos:

1.  Descarga el dataset desde Kaggle:
    [Road Traffic Dataset](https://www.kaggle.com/datasets/arashnic/road-trafic-dataset/data)
2.  **Importante:** Debes utilizar el archivo identificado como **`raw`**.
3.  Para entender en detalle el significado de cada columna, consulta la documentación oficial (específicamente la sección **iii. Raw manual counts data**):
    [Documentación de Metadatos (PDF)](https://storage.googleapis.com/dft-statistics/road-traffic/all-traffic-data-metadata.pdf)

## 2. Instalación

Se recomienda crear un entorno virtual específico para este proyecto para evitar conflictos de versiones.

Instala las dependencias necesarias ejecutando:

```bash
pip install -r requirements.txt
```
